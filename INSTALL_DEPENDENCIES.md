# Required Dependencies Installation

## 📦 Missing Dependencies

To use the new Add Class feature with the side panel (Sheet component), you need to install:

```bash
npm install @radix-ui/react-dialog
```

This is required for the Sheet component (side panel) to work properly.

## ✅ Already Installed

The following dependencies are already part of the project:
- react-hook-form (for form handling)
- date-fns (for date formatting)
- lucide-react (for icons)
- @radix-ui/react-label (for form labels)

## 🚀 Quick Install

Run this command in your terminal:

```bash
cd d:\Pronttera\FitStack\fitstack-admin-app
npm install @radix-ui/react-dialog
```

After installation, the Add Class side panel will work perfectly!
